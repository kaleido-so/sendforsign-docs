---
title: "Update user"
description: "Update existing user."
---

Updating a user, use the API key from the personal account in the header.

### Request

<ParamField body="action" type="string" required>
  Action type.
</ParamField>

<ParamField body="clientKey" type="string" required>
  ID of the client.
</ParamField>

<ParamField body="user" type="string" required>
  Structure, use the fields that should be updated.

  <Expandable title="child attribute">

  <ParamField body="userKey" type="string" required>
    ID of the user.  
  </ParamField>

  <ParamField body="fullname" type="string">
    First and second names of the user.  
  </ParamField>

  <ParamField body="email" type="string">
    Email of the user.  
  </ParamField>

  <ParamField body="customKey" type="string">
    User custom key, if you used it when creating the user.
  </ParamField>

  </Expandable>

</ParamField>

### Response

  <ResponseField name="user" type="string">
    User structure.

  <Expandable title="child attribute">

  <ResponseField name="changeTime" type="string">
    Change time.
  </ResponseField>

  <ResponseField name="userKey" type="string">
    ID of the user.
  </ResponseField>

  </Expandable>

  </ResponseField>

<ResponseField name="code" type="string">
  The code of the answer.
</ResponseField>

<ResponseField name="message" type="string">
  The message of the answer.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST 'https://api.sendforsign.com/v1/user' \
     -H 'X-Sendforsign-Key: YOUR_API_KEY' \
     -H 'Content-Type: application/json' \
     -d '{ 
		  "data": {
				   "action": "update",
				   "clientKey": "CLIENT_KEY",
				   "user": {
							  "userKey": "USER_KEY",
							  "fullname": "John Johnson",
							  "email": "john_johnson@mail.com", 
							  "customKey": "YOUR_COMBINATION"
						    }
				  }
       }'
```
</RequestExample>

<ResponseExample>
```json Response
	{
       "user": {
                "changeTime": "2023-12-01 12:18:55",
                "userKey": "USER_KEY", 
		},
       "code": "201",
       "message": "User updated"
    } 
```
</ResponseExample>
