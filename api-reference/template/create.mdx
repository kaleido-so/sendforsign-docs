---
title: "Create template"
description: "Create a new template using template value."
---

### Body parameters 

<ParamField body="action" type="string" required>
  Action type.
</ParamField>

<ParamField body="userKey" type="string">
  ID of the user who create the template. If not specified, the default user credentials will be used.
</ParamField>

<ParamField body="template" required>
  Template structure.

  <Expandable title="child attribute">

  <ParamField body="name" type="string">
    Name of the template.  
  </ParamField>

  <ParamField body="value" type="string">
    Content of the template, use HTML.  
  </ParamField>

  </Expandable>

</ParamField>

### Response

  <ResponseField name="template" type="string">
    Template structure.

  <Expandable title="child attribute">

<ResponseField name="createTime" type="string">
  Create time.
</ResponseField>

<ResponseField name="templateKey" type="string">
  ID of the template.
</ResponseField>

  </Expandable>

  </ResponseField>

<ResponseField name="code" type="string">
  The code of the answer.
</ResponseField>

<ResponseField name="message" type="string">
  The message of the answer.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST 'https://api.sendforsign.com/api/template' \
     -H 'X-Sendforsign-Key: YOUR_API_KEY' \
     -H 'Content-Type: application/json' \
     -d '{ 
		  "data": {
				   "action": "create",
				   "clientKey": "YOUR_CLIENT_KEY",
                   "userKey": "USER_KEY", 
				   "template": {
							  "name": "New template",
							  "value": "<p>Template text</p>"
						     }
				  }
       }'
```
</RequestExample>

<ResponseExample>
```json Response
     {
       "template": {
                "createTime": "2023-12-01 12:18:55",
                "templateKey": "TEMPLATE_KEY"
                 },
       "code": "201",
       "message": "Template created"
    }
```
</ResponseExample>
